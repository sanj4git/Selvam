## Objective

Build a functional finance tracking platform with user authentication, expense tracking, asset & liability management, and a basic financial dashboard.

---

## Features Covered

- User authentication (JWT-based)
- Expense tracking (CRUD)
- Manual asset entry
- Liability tracking
- Net worth calculation
- Basic dashboard with charts

---

## Technical Scope

### Backend

- Node.js + Express
- MongoDB for data storage
- RESTful APIs
- JWT authentication

### Frontend

- React (Vite)
- Basic forms and tables
- Dashboard charts using Recharts

---

## Deliverables

- Authenticated user flow
- Working APIs for expenses, assets, liabilities
- Net worth calculation logic
- Dashboard displaying financial summary

---

## Out of Scope (Phase 1)

- Live market APIs
- Family roles
- Alerts & notifications
- AI-based insights
